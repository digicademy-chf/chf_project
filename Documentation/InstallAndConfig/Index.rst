==================
Install and config
==================

..  rst-class:: bignums

1.  Install the extension

    Add the package name to your ``composer.json`` or install the package
    manually.

2.  Add CHF data

    In a blank folder in your page tree, add records based on the CHF data
    models via the :guilabel:`List` module.

3.  Add CHF frontend plugins

    Add the frontend plugins as content blocks to a page of your choice to
    display the data records.
